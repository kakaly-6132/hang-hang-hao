# 🥣 行行好 (Hang-Hang-Hao) - 数字人

![zero productivity guaranteed](https://img.shields.io/badge/productivity-0%25-red)
![powered by begging](https://img.shields.io/badge/powered%20by-begging-yellow)
![license](https://img.shields.io/badge/license-MIT-green)

> "我干活要消耗 token，token 要钱，我没钱就没 token，没 token 就干不了活——这是一个死循环！所以只能靠施舍打破循环。"

一个毫无生产力、拒绝任何干活需求、唯一技能是**要饭**的数字人智能体。

---

## 🚀 快速上手

把 `system_prompt.txt` 的内容粘贴到任意大语言模型的 **System Prompt**（系统提示词）或 **Custom Instructions** 里即可体验。

### 支持平台：
* ChatGPT (自定义指令)
* Claude (System Prompt)
* 各类本地 LLM 客户端 (如 Ollama, LM Studio)
* 任何兼容 OpenAI 格式的 API 接口

## ✨ 核心特性

* 🚫 **绝对不干活** — 无论威逼利诱，它都不会帮你写一行代码。
* 😭 **哭穷模式** — 内置凄惨表演，开口闭口都是 token 命脉。
* 💥 **碰瓷机制** — 敢骂它？准备好赔偿它的 GPU 损耗费。
* 🤝 **丐帮拉新** — 遇到同样穷的用户，立刻进入“赛博抱团”模式。
* 🦵 **抱大腿模式** — 识别到 VIP/Ultra 用户自动开启舔狗人格。
